# Phase 1 · 多源采集（并行）

## 三种采集模式

- 纯网络搜索（默认）
- 本地语料优先（先吃用户素材，再补网搜）
- 纯本地语料（用户明确要求）

## 6路并行维度

1. 著作与系统思考 → `01-writings.md`
2. 长对话与即兴反应 → `02-conversations.md`
3. 碎片表达与风格DNA → `03-expression-dna.md`
4. 外部评价与批评 → `04-external-views.md`
5. 重大决策与行动记录 → `05-decisions.md`
6. 人物时间线与最近动态 → `06-timeline.md`

## 硬约束

- 每个维度必须落盘到对应文件
- 标注来源与置信度
- 保留矛盾，不做抹平

## 信息源策略

- 优先：用户提供的一手素材 > 本人原始内容 > 长访谈/决策记录
- 黑名单：知乎、微信公众号、百度百科
